# Simple Interest Calculator

## Project Overview

The **Simple Interest Calculator** is a web-based application developed using **HTML, CSS, and JavaScript**. It allows users to calculate simple interest by entering the principal amount, rate of interest, and time period.

The application provides a simple and user-friendly interface for performing quick interest calculations.

---

## Features

* Easy-to-use interface
* Calculates Simple Interest instantly
* Responsive design
* Input validation through numeric fields
* Built using pure HTML, CSS, and JavaScript

---

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

---

## Formula Used

Simple Interest (SI) is calculated using:

SI = (Principal × Rate × Time) / 100

Where:

* Principal = Initial amount of money
* Rate = Rate of interest per year
* Time = Time period in years

## How to Run the Project

1. Download or clone the project files.
2. Open the project folder.
3. Double-click the `index.html` file or open it in any web browser.
4. Enter:

   * Principal Amount
   * Rate of Interest
   * Time (Years)
5. Click the **Calculate** button.
6. The calculated Simple Interest will be displayed on the screen.

---

## Sample Input

Principal Amount: 10000

Rate of Interest: 5

Time: 2 Years

### Output

Simple Interest = 1000
